## What is PM-OS?

PM-OS combines two powerful systems:

**PM Brain (Memory Layer)**
- Markdown-based second brain
- Tracks hypotheses with evidence
- Logs decisions with reversal conditions
- Maintains stakeholder context
- Weekly automated maintenance

**PM Skills (Workflow Layer)**
- 68 domain-specific skills
- 42 guided workflows
- 9 plugin domains
- Proven PM frameworks encoded

Together, they provide a complete operating system for product management work.

Quick install:
```bash

claude plugin install pm-all@pm-os
```
