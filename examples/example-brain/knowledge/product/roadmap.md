# Roadmap

> What's next, what's parked, what's killed. Status snapshot — full feature detail lives in [features/](./features/).

## Now

Active features being built or measured:

- **[incident-copilot-v1](./features/incident-copilot-v1.md)** — AI assistant with live summaries, troubleshooting steps, similar-incident retrieval, auto-postmortem drafts. Sprint 4 of 6. **Highest priority.**
- **[enterprise-rollout](./features/enterprise-rollout.md)** — SAML SSO improvements, RBAC, onboarding checklists, admin analytics. In development. Reduced allocation behind Copilot.
- **[slack-v3-integration](./features/slack-v3-integration.md)** — Interactive incident commands, threaded updates, automatic channel summaries. Design complete, engineering started.
- **[incident-analytics-redesign](./features/incident-analytics-redesign.md)** — MTTR, incident trends, team performance, custom reporting. UX validation with pilot customers.
- **[platform-reliability-q3](./features/platform-reliability-q3.md)** — API latency, search performance, service reliability. Ongoing, ~20–25% engineering capacity.
- **[smart-postmortem-generator](./features/smart-postmortem-generator.md)** — Shipped 3 weeks ago (2026-06-20 est.). Measuring: 42% WAU, 38→14 min creation time, ~30% manual editing. Root cause accuracy is current quality gap. *(Status: measuring)*
